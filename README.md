## 進め方

1. 自分の名前のブランチ名のブランチを作成する
1. 1章読み終わった段階でPRを作成、PRのタイトルは「rails_tutorial 名前」とする　※レビュー依頼は出さない
1. 章毎にコミットを分けて、上記2 で作ったPRに上げていく
1. チャットワークの日報にPRのリンクを記載する
